  <h1>🏦 Bank Client System (OOP)</h1>
  <p>C++ Console-Based Banking Management System</p>
</header>

<section>
  <h2>📌 Description</h2>
  <p>
    A simple bank management system developed using Object-Oriented Programming (OOP) in C++.
    The system supports clients, users, permissions, money transfers, and transaction tracking
    in a console-based environment.
  </p>
</section>

<section>
  <h2>🚀 Features</h2>

  <div class="feature-box">
    <h3>👥 Client Management</h3>
    <ul>
      <li>Add new clients</li>
      <li>View client information</li>
      <li>Update client data</li>
      <li>Delete clients</li>
      <li>Search for clients by account number</li>
    </ul>
  </div>

  <div class="feature-box">
    <h3>👤 User Management</h3>
    <ul>
      <li>Add system users</li>
      <li>View users list</li>
      <li>Update user data</li>
      <li>Delete users</li>
      <li>Assign permissions to users</li>
    </ul>
  </div>

  <div class="feature-box">
    <h3>🔐 Permissions System</h3>
    <ul>
      <li>Role-based access control (RBAC)</li>
      <li>Client management permissions</li>
      <li>Transfer permissions</li>
      <li>Transaction access control</li>
      <li>Admin-only user management</li>
    </ul>
  </div>

  <div class="feature-box">
    <h3>💸 Money Transfer System</h3>
    <ul>
      <li>Transfer between client accounts</li>
      <li>Balance validation</li>
      <li>Overdraft protection</li>
      <li>Secure permission-based operations</li>
    </ul>
  </div>

  <div class="feature-box">
    <h3>📊 Transactions System</h3>
    <ul>
      <li>Deposits, withdrawals, transfers</li>
      <li>Transaction history per client</li>
      <li>System audit trail</li>
    </ul>
  </div>

  <div class="feature-box">
    <h3>🖥️ Console Interface</h3>
    <ul>
      <li>Menu-driven system</li>
      <li>Simple navigation</li>
      <li>Role-based UI menus</li>
    </ul>
  </div>
</section>

<section>
  <h2>🛠️ Technologies Used</h2>
  <span class="tag">C++</span>
  <span class="tag">OOP</span>
</section>

<section>
  <h2>📂 Project Structure</h2>
  <ul>
    <li><b>main.cpp</b> → Main program file</li>
    <li><b>Bank.cpp / Bank.h</b> → Core system logic</li>
    <li><b>User.cpp / User.h</b> → User management & permissions</li>
    <li><b>Client.cpp / Client.h</b> → Client operations</li>
    <li><b>Transfer module</b> → Handles money transfers</li>
    <li><b>Transaction module</b> → Handles transaction records</li>
  </ul>
</section>

  </ul>
